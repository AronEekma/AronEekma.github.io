Iets meer over JavaScript en HTML

Er is een duidelijk verschil tussen Javascript en HTML. HTML heeft meer te maken met lay-out en hoe dus iets eruit ziet(website / app). JavaScript heeft meer te maken met functies om iets wat te laten uitvoeren. Hieronder ga ik het wat beter uitleggen.

Eerst ga ik maar over HTML hebben. HTML is een computertaal die wordt gebruikt om een website vorm te geven. Dit kan via plaatjes, tekst, achtergrond, menu's en nog veel meer. HTML is een een markup taal (opmaak taal), dit betekend dat het gemaakt is doormiddel van tags.
Tags zijn kleine dingen die zeggen waar iets begint en eindigt.

Een voorbeeld van een tag is <p> en </p>. Een < geeft het begin van een tag aan en een > geeft het einde van een tag aan. De tekst er tussen in, in dit geval 'p' geeft de tag een naam en wordt meestal gebruikt om dus aan te geven waar de tag mee bezig is. 'p' wordt heel vaak gebruikt om een paragraaf aan te geven. Maar er zijn ook andere tags die aangeven wat er met de tekst gebeurt, <strong> wordt gebruikt om een tekst dikgedrukt te maken.

Alle tags beginnen moeten beginnen met een letter of woord en <> er omheen. Daarna komt dezelfde tag maar dan een / voor de letter of naam. Dit is de eind tag. <p>Hallo</p> Als je zelf wilt zien hoe het in elkaar zit kan je CRTL + SHIFT + i doen. Daarna klik je op het elements knopje bovenaan het witte vakje dat zojuist is geopend.
